# web-application

# Clone repository về máy

    git clone https://github.com/phuongduclan/web-application.git

# Tạo nhánh mới

    git checkout -b feature/

# Hướng dẫn dẫn chạy chương trình

    Bước 1: Giải nén tập tin vào C:\xampp\htdocs\

    Bước 2: Chạy file script.sql để tạo database và dữ liệu

    Bước 3: Cấu hình file Database.php với connect string khớp máy người dùng
